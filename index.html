meallists.com

A meal planning website.

Meallists.com allows users to store their favorite recipes and use them to make meal plans. After selecting the desired meals for a given shopping trip, a ready-to-print grocery list is generated.

This full stack website was built using Angular 1.6. It is styled using Sass. The backend is run with node using express and massive. The database is done with SQL.

Alpha Clothing Clone

A clone of the alphaclothing.co website.

This clone of the alphaclothing.co website was done as a group project with two other developers. My contributions included animations, routing structure, product filtering, transfer of data across views with redux, and credit card payment.

lyricscanner.com

A website that checks lyrics for offensive content.

Lyricscanner.com is used to check lyrics for potentially offensive content. It takes a user inputted song and searches multiple lyric websites to identify the lyrics. From here is evaluates the lyric for key words that might be vular, sexual, or otherwise offensive.

Lyricscanner.com was build using PHP, MySQL, AJAX and javascript. The front end was made using Wordpress. Most of the logic and functionality is done with PHP.

  • Date: Nov 2014
  • Categories: PHP

IP Network Tester

Tester for your local network speed

This tool is used to test LAN speed by measuring the connection between two computers within the network. First, the application is downloaded and started on each computer being tested. Then, from either computer, you run a connection test. The results are saved on my external database. All past results are easily accessible by the click of a button.

The heart of this application is run by Node. The interface was done in Angular and the database is done with PostgreSQL. The Iperf program is used to run the data transfer.

  • Date: April 2017
  • Categories: Node, Angular
